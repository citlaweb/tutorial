Tutorial

inicia aqui

descargar el programa atom, crear una carpeta, arrastrar la carpeta al programa atom, crear los archivos html y css

html y css basico

etiquetas html, head, body, doctype, lang, meta charset, title. vincular los estilos etiqueta link,

titulo y menu

etiquetas nuevas: display flex, justify content space beetween, font size, list style, margin right, align items...agregar el tipo de fuente desde google fonts, la que tengo es poppins

titular y sub + boton

etiquetas nuevas: h1, a, text-decoration none, margin-top, background, padding, margin-bottom, color, header, height 100vh, background-size cover, max width, width, margin auto. recomendacion como ayuda utilizar el modo consola desde el navegador.

comentarios

Tecnologias

carousel de javascript

owlcarousel

buscar en google owlcarousel, descargar el archivo, crear una carpeta nueva con el nombre owlcarousel, copiar los 3 archivos siguientes:owl.carousel.min, owl.theme.default.min y owl.carousel.min este es js

jquery

buscar en google jquery, clic en el que dice compressed, ctrl + a luego ctrl + c, desde atom creo nuevo archivo, y lo guardo como jquery.min.js

codigos

desde la web de owlcarousel ir a docs, installation, copiar codigo css y lo pego en head. copiar codigo js y verifico la ruta esto lo pego antes de cerrar el body. copiar codigo html y creo el section, cambio texto por imagenes. ir a demos, basic copiar codigo setup en un script despues del codigo js, cambio codigo de items de mostrar en pc por el numero 4. agregar codigo css para dar estilo a las flechas.
